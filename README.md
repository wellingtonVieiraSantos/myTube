# MyTube

## Projeto Simples de um player de video estilo Youtube, personalizado e sem o uso dos 'controls'.

### Algumas features implementadas:

- Volume controler
- Play and pause
- Timer
- Playback speed control
- Mini player
- Theater mode
- Fullscreen mode
- Responsiveness
- Shortcut keys: 
  - Space-bar and 'k' : 'Play and pause'
  - 'f' : 'Fullcreen toggle'
  - 't' : 'Theater mode toggle'
  - 'i' : 'Miniplayer mode'
- more soon...


### Tecnologias usadas:

- React.js
- TailwindCss
- React icons
- Vite
